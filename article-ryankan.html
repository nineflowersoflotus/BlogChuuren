Eficiência de Blocos: Ryankan

Ryankan se trata de uma estrutura onde temos dois kanchan grudados compartilhando a mesma peça (ex.: 4p6p8p). Seu ukeire é o mesmo de uma espera ryanmen (8), com o detalhe que, se tivermos um iishanten com duas esperas, uma delas ryankan, se completarmos a outra espera sempre teremos uma espera kanchan em tenpai. A vantagem disso é a construção de uma armadilha de suji (ex.: 1m3m5m -> 1m3m, 2m parece seguro pelo descarte de 5m).

Fragmentos de Ryankan

Quando temos uma forma de duas peças como 1p5p, chamamos isso de um fragmento de ryankan, pois a compra de um 3p forma um ryankan. Conhecer essa forma é útil em decisões como escolher entre descartar 1s6s ou 1p5p, mas no longo prazo acabarão sendo descartadas de qualquer forma caso não sejam formadas rapidamente.
